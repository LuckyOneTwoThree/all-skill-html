## 1. 产品概述

All-Skill Galaxy 是一个交互式产品工程 Skill 全景可视化介绍网页，展示 179 个 AI Agent Skill 的全局架构、数据流向和层级关系。面向产品经理、设计师、工程师和开源社区贡献者，让他们一眼理解这套产品工程方法论体系的完整结构和设计理念。

- 核心目标：让访问者在 3 分钟内理解四大领域的全局关系和核心数据契约流
- 差异化：不是简单的文档站，而是一个可交互探索的「产品工程星系图」

## 2. 核心功能

### 2.1 页面模块

1. **全景图谱页**：力导向交互图谱，展示全部 179 个 Skill 的层级关系和数据流向
2. **数据契约流页**：Sankey 流向图，展示跨领域数据契约（PRD、OpenAPI、设计令牌等）的传递路径

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 全景图谱页 | Hero 区域 | 大标题 + 核心数据统计（179 Skills / 4 领域 / 39 编排器）+ 向下引导 |
| 全景图谱页 | 交互图谱 | D3.js 力导向图，三层视图切换（领域/模块/完整），节点可点击展开详情面板 |
| 全景图谱页 | 详情面板 | 右侧滑出面板，展示节点基本信息、定位、输入输出、上下游关系、质量检查 |
| 全景图谱页 | 筛选搜索 | 顶部搜索栏 + 领域/类型/模块三级筛选 |
| 全景图谱页 | 数据契约流 | Sankey 图展示 PRD/OpenAPI/设计令牌等跨领域数据流动 |
| 全景图谱页 | 模块卡片 | 四大领域模块卡片网格，可筛选浏览 |
| 全景图谱页 | 页脚 | GitHub 链接、MIT License、贡献指南 |

## 3. 核心流程

用户进入页面 → 浏览 Hero 区域了解全局概况 → 滚动到交互图谱区域 → 通过三层视图切换从宏观到微观探索 → 点击节点查看详情 → 切换到数据契约流视图理解跨领域数据传递 → 浏览模块卡片获取结构化信息

## 4. 用户界面设计

### 4.1 设计风格

- **美学方向**：Editorial Data Viz — 编辑式数据可视化风格，将信息图表的严谨与杂志排版的优雅结合
- **主色**：#4F46E5（Indigo-600）— 智慧、专业、科技
- **辅助色**：#7C3AED（Violet-600）— 创意、设计
- **CTA/强调色**：#06B6D4（Cyan-500）— 数据流动、交互反馈
- **背景色**：#FAFBFC — 极浅暖灰，避免纯白的刺眼感
- **卡片色**：#FFFFFF + 微妙渐变边框
- **文本色**：#0F172A（Slate-900）
- **次要文本**：#64748B（Slate-500）
- **四大领域色**：PM=#6366F1 / UI=#8B5CF6 / Backend=#0EA5E9 / Cross=#F59E0B
- **按钮风格**：圆角8px，Indigo填充，hover时微妙上移+阴影加深
- **字体**：标题 Sora（几何感、未来感），正文 DM Sans（清晰、现代），中文 Noto Sans SC
- **布局风格**：全宽图谱区域 + 浮动详情面板 + 卡片网格
- **图标**：Lucide Icons，线性风格，1.5px描边

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 全景图谱页 | Hero 区域 | 渐变背景（Indigo→Cyan微妙渐变），大标题 Sora 700，统计数字动画计数，向下箭头呼吸动画 |
| 全景图谱页 | 交互图谱 | 浅灰背景，节点按层级大小/颜色区分，连线带流动粒子动画，右侧380px详情面板滑出 |
| 全景图谱页 | 数据契约流 | Sankey 图，绿色流动线，hover显示契约详情tooltip |
| 全景图谱页 | 模块卡片 | 白色卡片+左侧领域色竖线，hover上移4px+阴影，网格布局 |

### 4.3 响应式

- 桌面优先（1440px+），平板（768-1439px）图谱全宽+覆盖式详情面板，移动端（<768px）切换为卡片列表视图

### 4.4 动效设计

- 页面加载：Hero 文字逐行淡入上移（stagger 100ms），统计数字 count-up 动画
- 图谱节点：hover 放大1.15x + 光晕效果，点击时脉冲波纹
- 数据契约流：粒子沿路径持续流动，2s循环
- 详情面板：translateX 滑入 300ms ease-out
- 视图切换：节点位置平滑过渡 500ms
- 搜索高亮：非匹配节点 opacity 过渡 200ms
- 卡片 hover：上移4px + 阴影加深 200ms ease-out
- 背景：Hero 区域有微妙的渐变流动动画（20s循环）
